Preguntas Frecuentes

¿Se pueden combinar envío, cuotas y regalo? ↓

Sí. Puedes activar las tres reglas simultáneamente. El widget se encargará de mostrar una barra de progreso separada y elegante para cada beneficio activo, permitiendo al usuario ver todo lo que puede ganar sumando productos.

¿Cuándo se actualizan las barras? ↓

Se sincronizan en tiempo real. Cada vez que el usuario agregue, elimine o modifique la cantidad de un artículo en el carrito, la barra recalculará los totales instantáneamente y reproducirá las animaciones correspondientes.

¿Qué significa "Aplica a"? ↓

Te permite restringir la promoción. Por ejemplo, si seleccionas "Aplica a: Categoría Zapatillas", la barra de Envío Gratis solo sumará el dinero gastado en zapatillas. Si el usuario compra remeras, ese dinero no sumará para la meta de envío.

Tips CRO para aumentar el ticket ↓

Recomendamos fijar la meta de "Envío Gratis" entre un 15% y un 25% por encima del Ticket Promedio actual de tu tienda. Utiliza los colores de tu marca y asegúrate de que el texto sea breve (Ej: "Agrega $X más para Envío Gratis").
