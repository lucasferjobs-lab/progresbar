Preguntas frecuentes

Guía rápida para operar la app y mejorar rendimiento.

¿Se pueden combinar envío, cuotas y regalo?

Sí. Puedes activar las tres reglas y el carrito mostrará una barra por cada beneficio activo.

¿Cuándo se actualizan las barras?

Se actualizan automáticamente con cada cambio en el carrito (agregar, quitar o cambiar cantidades).

¿Qué significa "Aplica a"?

Define qué subtotal se usa para calcular el progreso: todo el carrito, un producto o una categoría.

Tips CRO para aumentar el ticket promedio

Combina envío gratis con cuotas, usa metas alcanzables y textos cortos y claros.
